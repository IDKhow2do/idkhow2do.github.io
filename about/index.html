嗨，我是龙山

现居成都，正努力从一名数字集成电路专业的学生，成长为能把 HDL 写顺手、也能把生活过出点“花活儿”的人。
朋友叫我“一夔”或者“小龙山”，在这儿我给自己留了个小窝，专门放那些从周报、课程笔记到碎碎念都不想丢的东西。

写博客对我来说像记工时：

  • 有一段时间沉迷 The Missing Semester、CS61A，于是把网课笔记、踩坑记录全写了下来，方便以后掉进同一个坑还能自救。
  • 也试图把数字 IC 的学习路径梳理成条理，比如《Digital IC Learning》《hdlbits》的心得，偶尔夹带一些 Verilog 的碎碎唠叨。
  • 每周的“周报”是对生活的复盘：考试、焦虑、旅行、追番、装机、与朋友夜聊，全都混在一起，像是给未来的自己留下一点线索。
  • 娱乐向也不少——喜欢蜘蛛侠、《进击的巨人》、各种丧丧的歌，还会安利电影、书、工具，想起来了就写。

现在在折腾的事

  • 把基础打牢：补数学、补物理、补那些被时间偷走的概念，朝“能独立完成项目”的状态靠近。
  • 写点好玩的代码：从 whisper 到图床脚本，再到这个照片墙，和命令行小玩具，堆起来就是自己的工具箱。
  • 记录真实情绪：博客没有“人设”，可能上一秒还在讲学习，下一秒就开始吐槽 GFW 或为一部动画眼睛发光。
  • 周报随缘：曾经野心勃勃想周周总结，现实是经常摸鱼，更惨的是现在工作后真·周报按时交，反而更想来这儿写点“只给自己看的废话”。这也算是一种宣泄平衡吧。
  • 保持好奇心：不保证更新频率，但能保证一旦找到有意思的东西，就会写下来。

这个博客

  • 名字叫“一夔的小窝”，源于那句“在互联网上占一席之地”。
  • 主题是 Hugo LoveIt，换上了霞鹜文楷，还能自动切换明暗模式。
  • 有个 3D 悬浮的照片墙，记录我“随手咔嚓”的瞬间。
  • 评论区用 Gitalk，想聊就登陆 GitHub 留个言。
  • 文章是多语言混写的，看心情可能是中文、英文或者更随意的表达。
