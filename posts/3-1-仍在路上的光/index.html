2025 年即将过去，现在是 11.11 夜，我决定重启我的周报，可能会比较简短，但我就是不想有什么压力，想随心所欲的，自由自在的，随便下去…

没上进心（可能有人会这么说）

引子

考驾照的时候，科目二我考试的时候没过，但我当时其实觉得自己学得已经OK了。距离下一次考试需要隔很长的时间，一般都是安排的先考科目三。所以说，在练习科目三的那段时间，我报名了科目二的考试，因为他们觉得来不及考。我算了一下时间，在我开学前的前两天，那个时候我还在家里，我认为可以偷偷地去考一下。在那段时间，教练天天带我去练科目三。

科目三第一次是起步的时候没有看后方来车，被扣掉了一百。第二次是前面堵车了之后，我直接左转到了左边，我想着这样开会快一点，结果语音让我在左车道上换到左车道，又挂掉了。

当时的心情非常的沮丧，因为自己相当于考了两次，两次都没过。不太好意思面对家长和教练。他们都会说这种东西确实就是比较考验运气，没办法，但是我知道没过肯定会很难受会心情不好。后面我偷偷地直接去把科二考了，他们当时都很开心。但说我厉害没去练科二居然都过了，但也会有合理化的说法说练了科三之后再练科二，确实会很有帮助，不可否认，应该确实是有帮助的。但是我认为自己可能在第一次考科二的时候，就已经可以通过了。我是想把这个当做一个引子来讲。

我常常认为自己是一个闷声干事的人，如果我向外输出了，我很有可能坚持不下来，或者说我因为有压力而开始逃避，我喜欢那种没有压力的环境下的努力，自然而然的进步。就像是大家都在卷的时候，我看自己的计算机相关的书籍，培养爱好，好像在中国卷就是一种错误，精进自己的专业就是一种错误，除非你能够抵抗他人的捧杀，而我抵抗不住。我只要被讲什么卷，我就马上想要放弃，因为我不想被人看到自己努力，然后还一事无成，哪怕只是有这个风险。

前段时间偷偷报名了考研，虽然自己并没有怎么学习，但是还是在报名截止日期的最后一天报名了，想着哪怕只是一个月可以冲冲看，看看自己什么水平，然后在准备网上确认的时候，发现截止日期是昨天，哈哈哈。

也是，大学考试的时候还可以冲一冲，但是面对这种大型考试，怎么可能是冲一冲就能好的，而我的动力也好像就只是可以冲一冲而已。

但是因为没有网上确认，相当于自己接下来一段时间的任务都不知道做什么了，花了两天的时间来想(实则是刷抖音混时间)，在睡前总觉得自己不能什么都不做吧，于是先把博客捡起来再说吧，还是有挺多想做的事情的，且不论能否做到，但是还是有很多想做的未做的事情的，这个放在结尾再说吧。

没有上进心

最近，被说没有上进心了

  • 没有上进心
  • 瘦猴
  • 挣的钱不够养家
  • 不积极抗痘(喜甜食，一段时间就要喝奶茶) 我是个不太能接受别人批评的人，可能因为自尊心比较强，或者是比较自卑，甚至交朋友也是避免跟那种有权势或者有钱的人多接触，可能会刺激到自己的自尊心？

对于伴侣来说，我希望对方是一个完全接纳对方，或者说至少在谈的过程中，不需要因为担心对方不喜欢自己而担心，不需要担心对方讨厌自己的什么缺点，或者哪怕是缺点也是接纳，这样才会给我以安全感。

  1. 没有上进心，因为我不想被人看到我上进，我不想被人知道我上进了结果没上进。
  2. 瘦猴，就是长不胖，但我不觉得自己的力气如何小，至少在同样重量级里面，而且锻炼身体的事情，自己一直有在规划，可能没在执行。
  3. 挣的钱够我现在花，我没办法一下子挣很多钱，以后会慢慢变多的吧，但确实跟名校毕业的没法比，目前暂时只有这个实力。
  4. 心情不好，甜食可以治愈，常提喝奶茶更像是一种社交属性，但其实公司里约我喝奶茶，我基本都是拒绝的，说自己在戒糖，反而我认为熬夜才是我最该改的，现在又是深夜了，1:37

我常常认为在一段恋爱关系中，我是属于那种对对方特别依赖的人。如果我没有这么一段可以值得让我依恋的感情，可能会很没有安全感，我会到处找人分享以缓解。可这也可能恰恰就是为什么我谈恋爱都会谈的比较久。因为我对于我认定的关系都会有很强的依赖性，除非遇到了什么原则性的错误。

一个关于我因噎废食的证据，昨天晚上突发奇想看书睡觉，发现 kindle 的默认字体有点丑，想安装一个漂亮的字体，比如现在这个，然后发现自己的microUSB 连上去只能充电不能传输数据，于是网购质量更好的数据线并决定更换字体之后再开始看书。

物欲

心情不好就想购物，最近心情不好，看看我有多少东西想买的：

  • 相机，本来预算就 6000 左右，但现在看着看着一算就已经上万了，舍不得买，暂时不买了，因为攒钱也有快感，要是买了相机，我攒钱的快感会丢失大半的；
  • 电脑，我也不知道为什么最近突然想买个台式电脑，哦好像是因为想串流，因为尝试过串流，发现延迟特别低，MAC 又不支持串流，反观 Windows 的效果特别好，想想随时随地玩家里的游戏，多爽的一件事情啊，但是最近内存太贵了，一点也不划算；
  • 最近把 macbook air 低价的方式转赠某人，我没有什么可以随时携出的电脑了，又想买一个长续航的电脑，可能会选择一个 windows 的全能本，但是没怎么看，因为上面两个东西我更想买一点

还有一个想法，就是想过年回家，把老家的东西全都拍下来，甚至举办一个摄影大赛，当然这些都是幻想而已，我这个 i 人，怎么可能举办的起来，我能想到的极限就是弄一个小程序，然后让大家可以上传自己的照片，然后投票，我给前几名的照片发个奖就够了，哈哈哈，不过真的好怀念老家的一切，原来出来打工了真的会想家，很奇怪，读书的时候就不会，可能因为家真的是温暖的港湾吧。

对了最近有看 EVA，本来以为会有很多剧集，但是好像也还好，暂时已经看到第十集啦，观感不错，特别精致的动画，而且很难想象是九十年代的动画。明日香和绫波丽都是没看之前，光看造型就已经很喜欢的角色了。

想做的事情

  • 继续学习 Linux
  • FPGA/CPU/Verilog(虽然现在的工作完全用不到这些，但是总觉得这些东西自己没搞透，还是想多研究研究，想知道是怎么运作起来的)
  • 如果有空的话，我还是希望自己多学习学习，不一定为了考研，但如果有空的话，也可以考研，当做 side project
  • 想去国外读书，哪怕是很久以后

说起来，为什么在考研最后一天选择还是报名，一方面就是因为如果明年再报名，哪怕是努力备考了，明年也差不多到了该跳槽的时候了，到时候本来就有一波工资的涨幅，可能大概就是研究生的水平的工资吧，所以想着如果从金钱上来看的话，即便是要考上也还是今年考上比较好，不然的话明年总觉得损失很大，但是造化弄人，或者说我自己本身也没有足够重视吧，现在就当做一个随缘的事情，随遇而安吧，就这样没有上进心的走下去吧。
